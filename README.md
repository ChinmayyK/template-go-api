# Official DevForge Go API Template

A minimal, production-ready Go 1.22+ API utilizing standard library routing (`net/http`) and a minimal Alpine Dockerfile.

## Quickstart
Do not clone this repository manually. Run the following command with the DevForge CLI:

```bash
devforge init my-app --template go-api-starter
```

*DevForge will automatically verify you have Go 1.22 and Docker installed before scaffolding this application.*
